# Extended style guide

This is a longer reference document that the main SKILL.md body
intentionally keeps out of its primary flow (progressive disclosure) —
only load it when a changelog entry needs closer style scrutiny.

- Prefer active voice: "You can now export..." not "Export is now
  possible..."
- Numbers: spell out one through nine, use digits for 10+.
- Never use exclamation points in changelog copy.
